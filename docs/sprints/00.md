# Sprint 00 — CLI/REPL with Fake LLM

## Goal

Build the basic user interface: a terminal loop that reads user input and returns a response.
The LLM is faked via a Go interface so Ollama is not needed to run this sprint.

## As a user

As an employee of the company, I want to open a terminal and ask a question about the company.
Even if the answer is fake for now, I want to see that the interface works and my input is received.

## Done when

- Running `go run .` shows a `>` prompt
- Typing a message prints a response from the fake LLM
- Typing `quit` exits the program cleanly
- Each turn is appended to an in-memory message history as `{role: "user", content: "..."}` and `{role: "assistant", content: "..."}`
- The full history is passed to the fake LLM on every call (fake LLM ignores it, but the plumbing is in place)
